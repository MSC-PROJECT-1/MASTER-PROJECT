# 🌾 Explainable Crop Recommendation System  
### *Balancing Predictive Performance with Interpretability*

---

## 📌 Overview

This project presents a **complete end-to-end machine learning pipeline** for building an **interpretable crop recommendation system**.

Unlike traditional approaches that focus only on predictive accuracy, this framework integrates:

- 🎯 **Predictive Performance**
- 🔍 **Model Explainability**
- ⚖️ **Real-World Usability**

The goal is to develop a system that is not only accurate, but also **transparent, interpretable, and trustworthy**.

---

## ✨ Motivation

In agriculture, decision-making requires more than predictions:

- Farmers need **clear explanations**
- Black-box models reduce **trust**
- Interpretability improves **adoption**

> ⚠️ High accuracy alone is not sufficient for real-world deployment.

---

## 🧠 Key Features

- ✔ End-to-end ML pipeline (EDA → Modelling → Evaluation → Explainability)
- ✔ Multiple model comparison
- ✔ Explainable AI integration (SHAP & LIME)
- ✔ Multi-objective model selection
- ✔ Advanced data visualisation
- ✔ Statistical validation

---

## 🏗️ Methodology

### 1. Exploratory Data Analysis (EDA)
- Feature distributions  
- Class balance analysis  
- Correlation heatmap  
- PCA for dimensionality insights  

---

### 2. Model Development

The following models were implemented:

- 🌳 Random Forest  
- ⚡ XGBoost  
- 📐 Support Vector Machine (SVM)  
- 🧠 Artificial Neural Network (ANN)  

---

### 3. Performance Evaluation

Models were evaluated using:

- Accuracy  
- Precision  
- Recall  
- F1-score  
- Cross-validation  

---

### 4. Explainable AI (XAI)

#### 🔍 Global Explainability
- SHAP values  
- Feature importance ranking  

#### 🎯 Local Explainability
- LIME explanations  
- Instance-level interpretation  

---

### 5. Multi-Objective Optimisation

A key contribution of this project is:

### 🧩 Multi-Objective Composite Score (MOCS)

This combines:

- Predictive performance  
- Explainability quality  

➡️ Ensuring selection of models that are both **accurate and interpretable**

---

### 6. Statistical Validation

- Friedman Test  
- Wilcoxon Signed-Rank Test  
- Cohen’s d (effect size)
